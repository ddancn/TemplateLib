# TemplateLib
Android模板工程

* 首页ViewPager+BottomNavigationView实现tab页功能
* BaseView基类（BaseActivity/BaseFragment）
    * 划分onCreate
    * 绑定ButterKnife
    * 自定义标题栏
    * 适配沉浸式状态栏
    * 防快速点击
    * 绑定BasePresenter
* 网络层基类
    * BaseResponse基本返回数据
    * BaseCallback基本回调，拆解数据，处理错误信息
* 工具类
    * 常用对话框
    * 其他工具...
* 常用第三方库引入
